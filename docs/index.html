〈使い方〉


1. 目的のパズルを選択し、計算ボタンを押します。計算に時間がかかる場合があります。


2. リストから目的の組み合わせを選択します。

    1.ソート：▲、▼ボタンをクリック - 最後のソート状態に基づいて追加ソートされます。

    2. フィルタ: 最小、最大数で範囲を調整します。


3. さまざまなバッチが存在する場合、上部からバッチを変更できます。


4. カスタムをチェックしたら、パズルプレートのマスをクリックして開閉できます。

    1.リフレッシュ：パズルプレートを配列に変換

    2. エクスポート: 配列 → tetro ファイル(コンピュータに個人的に保存)

    3.インポート：tetroファイル→適用（保存されたファイルを読み込む）

    4.適用：配列をパズルプレートに反映して計算
